### Get the project!
1. Clone and install.
```bash
$ git@github.com:jeescu/react-stack-app.git
$ cd react-stack-app
$ npm install
```

2. Remove `.git`, update `package.json` and make it your own!
```bash
$ rm -rf .git
$ git init
```

3. Start the app.
```
$ npm start
```
